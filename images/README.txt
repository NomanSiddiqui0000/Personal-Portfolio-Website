REQUIRED IMAGES FOR THE PORTFOLIO

Please add the following images to this directory:

1. profile.jpg - A professional headshot or portrait image (recommended size: 500x500px)
2. about.jpg - An image for the About section (recommended size: 600x800px)
3. project1.jpg - Image for the first project (recommended size: 800x600px)
4. project2.jpg - Image for the second project (recommended size: 800x600px)
5. project3.jpg - Image for the third project (recommended size: 800x600px)

Image Guidelines:
- Use high-quality, professional images
- Optimize images for web (compress to reduce file size)
- Use consistent aspect ratios for project images
- Ensure images are relevant to the content they represent
