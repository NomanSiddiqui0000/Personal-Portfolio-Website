A web scraping tool that extracts classified ads from OLX Pakistan across various categories, including Vehicles, Electronics, Property, Jobs, and more. The scraper navigates multiple pages, collects relevant details such as title, price, seller information, location, and description, and saves the structured data for further analysis.

Technologies Used:
  • Python (Scrapy, Requests)
  • Scrapy Framework
  • CSS Selectors & XPath
  • Pandas
Key Features:
  • Scrapes multiple OLX categories dynamically
  • Extracts structured information
  • Supports pagination for large-scale data
  • Saves data in CSV or JSON formats
  • Handles delays and retries

Rozee.pk Jobs Scraper

Project Overview

The Rozee.pk Jobs Scraper is a Node.js-based web scraper designed to extract job listings from Rozee.pk efficiently. Using Puppeteer for browser automation and Cheerio for HTML parsing, it enables users to scrape job postings across multiple pages, filter results based on job criteria, and export structured job data into CSV files for analysis.

Technologies Used:
  • Node.js & Puppeteer
  • Cheerio
  • CSV-Writer
Key Features:
  • Scrapes job listings from multiple pages
  • Extracts key job details
  • Supports filtering options
  • Handles pagination & dynamic delays
  • Exports structured data
